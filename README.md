Ce projet contient du code pour : 
- Extraire des polylignes via l'API Google Maps avec toutes les données associées au mode de transport prévu.
- Estimer la demande d'un quartier géoréférencé en croisant les polylignes le coupant pour en extraire cette demande
- Convertir une matrice d'une grande taille en une matrice plus petite via une table de correspondance (par exemple utile pour une matrice de demande Origine-destination de microzones qu'on souhaite transformer en matrice OD de macrozones avec moins d'OD)
