Ce projet contient : 
- Du code pour extraire des polylignes via l'API Google Maps avec toutes les données associées au mode de transport prévu.
- Du code afin de faire convertir une matrice d'une grande taille en une matrice plus petite via une table de correspondance (par exemple utile pour une matrice de demande Origine-destination de microzones qu'on souhaite transformer en matrice OD de macrozones avec moins d'OD)
